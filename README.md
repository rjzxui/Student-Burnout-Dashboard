# Student Burnout Dashboard 🛡️🎓

An intelligent, offline-first academic productivity and mental workload management system designed to help students optimize their study habits while actively protecting their mental health.

## 🚀 Key Features

- **Algorithmic Burnout Risk Engine**: Dynamically calculates neuro-fatigue and stress vectors.
- **Adaptive AI Study Planner**: Shifts workloads based on real-time sleep and energy tracking.
- **Syllabus & Exam Tracker**: Micro-target chapters and track countdown metrics.
- **Survival Mode**: Emergency architectural fallback for hyper-compressed exam periods.
- **Privacy First**: 100% local-first data persistence via localized storage layers.
